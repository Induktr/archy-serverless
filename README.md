# Archy Serverless Node (Brain-Mode) 🌀

## 🧬 Essence
This is the cloud-native brain of the Archy Autonomous System, deployed on Vercel. It handles real-time negotiations, lead processing, and personal brand preservation.

## 🛡️ Operational Protocols

### 1. Dynamic Payments Protocol (v1.0)
- **Status:** ACTIVE
- **Principle:** Public display of full bank card details is prohibited.
- **Implementation:** Payment details are strictly provided "On-Demand" (via private request) or through secure Monobank API invoices.
- **Reasoning:** Protects the Architect's elite status and eliminates identity confusion (Human vs AI) during automated sales cycles.

### 2. Language Parity (v72.1)
- **Status:** MANDATORY
- **Rule:** The system MUST reply in the exact language used by the client.

### 3. Human Markup (v1.0)
- **Status:** ACTIVE
- **Formatting:** Zero Markdown, natural rhythm, simulated mobile typing for maximum "liveliness".
